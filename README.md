# DeepSeek-TUI Nightly Builds

Automated nightly builds of [DeepSeek-TUI](https://github.com/Hmbown/DeepSeek-TUI) for Linux.

## Platforms

- **linux-x64** — x86_64-unknown-linux-gnu
- **linux-arm64** — aarch64-unknown-linux-gnu

## Binaries

- `deepseek` — CLI dispatcher
- `deepseek-tui` — Terminal UI engine

## Schedule

Builds run daily at UTC midnight, plus on-demand via `workflow_dispatch`.

## Download

Latest nightly builds are published as a [pre-release](https://github.com/douglarek/deepseek-tui-nightly/releases/tag/nightly) tagged `nightly`.
